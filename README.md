# 阅读资料库

这是我的日常学习资料库，也是“小阅”Agent 助手维护的个人 LLM Wiki 工作区。

## 仓库结构

- 顶层分类目录保存原始资料，例如 `计算机图书/`、`经济/`、`AI/`、`论文/`、`数学/`、`文章合集/`、`名人/`、`其它/`、`图片/`、`效能/`。
- `wiki/` 保存由 Agent 维护的知识层，包括来源页、概念页、综合页、问题回答和健康检查。
- `CLAUDE.md` 是完整维护规范。
- `AGENT.md` 是 Agent 快速入口。
- `llm-wiki.md` 是 Karpathy LLM Wiki 方法原文。

当前仓库没有 `raw/` 目录；后续摄入资料时，直接从顶层分类目录开始。

## 使用方式

1. 新增资料时，把文件放到合适的顶层分类目录。
2. 让 Agent 按 `CLAUDE.md` 的摄入工作流更新 `wiki/`。
3. 查询时先从 `wiki/index.md` 开始，再按双链进入来源页、概念页和综合页。
4. 周期性运行健康检查，处理断链、重复来源、缺失来源和待深度摄入队列。
