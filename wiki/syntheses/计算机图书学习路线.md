# 计算机图书学习路线

## 简短结论

这批书最适合支撑一条从“写好代码”，到“理解计算机系统”，再到“构建后端服务”和“运行分布式应用”的学习路径。

## 路线一：后端工程核心

1. 代码质量：`代码整洁之道`、`重构：改善既有代码的设计`、`编写可读代码的艺术`。
2. Java 深度：`Effective Java`、`Java并发编程的艺术`、`Java编程思想`、`Java核心技术`。
3. 框架与运行时：`Spring实战`、`SPRING技术内幕`、`Spring Boot实战`、`Netty实战`、`How Tomcat Works`。
4. 中间件：`Redis设计与实现`、`Redis开发与运维`、`高性能MySQL`、`MySQL技术内幕：InnoDB存储引擎`。
5. 架构：`企业应用架构模式`、`企业集成模式`、`微服务设计`、`大型网站技术架构`。
6. 运行与交付：`Docker从入门到实践`、`Kubernetes开源书`、`Observability Engineering`。

## 路线二：系统基础

1. 编程基础：`计算机程序的构造和解释`、`七周七语言`。
2. 计算机系统：`深入理解计算机系统`、`程序员的自我修养—链接、装载与库`。
3. 操作系统：`现代操作系统`、`UNIX环境高级编程`、`UNIX操作系统设计`。
4. 网络：`TCP_IP详解卷1：协议`、`UNIX网络编程卷1`、`HTTP权威指南`。
5. 编译与计算：`编译原理`、`计算的本质`。

## 路线三：架构与设计

1. 编程技艺：`Code Complete`、`程序设计实践`、`程序设计方法`。
2. 面向对象与设计模式：`设计模式：可复用面向对象软件的基础`、`Head First 设计模式`、`设计模式之禅`。
3. 重构与领域建模：`重构`、`领域驱动设计`。
4. 应用架构：`企业应用架构模式`、`Architectural Styles and the Design of Network-based Software Architectures`。
5. 集成与分布式实践：`企业集成模式`、`微服务设计`、`大型网站技术架构`。

## 路线四：移动端与客户端平台

1. Android 入门：`Android 第一行代码`、`Android群英传`。
2. Android 内部机制：`Android系统源代码情景分析`、`深入理解Android 卷I/卷2`、`深入理解Android内核思想`。
3. Android 工程实践：性能优化、安全、热修复、自动化测试、网络编程、音视频。
4. 跨平台：`Flutter技术解析与实战`、`Flutter企业级应用开发实战手册`、`kotlincn-docs`。

## 路线五：算法与面试

1. 基础：`算法导论`、`算法 第4版`、`数据结构与算法分析`。
2. 练习：`编程珠玑`、`编程之美`、`剑指offer`。
3. 友好入口：`hello-algo-1.1.0-zh-java`。

## 路线六：AI 与机器学习基础

1. 机器学习基础：`Machine-Learning 《机器学习（西瓜书）》`。
2. 深度学习实践：`d2l-zh-pytorch`。
3. 迁移学习：`Transfer-Learning迁移学习简明手册`。
