# 个人 LLM Wiki Agent 入口

这是本知识库的 agent 快速入口。完整规则以 `CLAUDE.md` 为准。

`CLAUDE.md` 定义了：

- 当前仓库结构；
- wiki 页面约定；
- 来源资料摄入流程；
- 知识库查询流程；
- wiki 健康检查和维护流程。

## 快速开始

1. 先读 `CLAUDE.md`。
2. 再读 `wiki/index.md`。
3. 查看最近操作：
   `rg "^## \\[" wiki/log.md | tail -10`
4. 新来源默认来自用户指定的顶层来源目录，例如 `计算机图书/`、`AI/`、`论文/`、`经济/`。
5. `raw/` 是新增文件收件箱，当前仅含 `.gitkeep` 占位符；如果用户没有指定来源目录，先检查这里是否有待摄入文件。
6. 不要把已归档顶层目录资料搬进 `raw/`，也不要擅自移动或删除 `raw/` 中的文件。
7. 不要修改来源目录中的原始资料，除非用户明确要求。
8. 只要 wiki 内容发生变化，就同步维护 `wiki/index.md` 和 `wiki/log.md`。

## Agent 默认行为

- 搜索优先使用 `rg`。
- wiki 页面之间使用 Obsidian 双链：`[[页面名]]`。
- 回答时引用 wiki 页面和来源文件路径。
- 优先做小步、可追踪、可继续积累的 wiki 更新。
- 不确定时，把不确定性写出来，不要假装已经证实。
